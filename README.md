# DiploVault – Secure Storage & File Management System

## Overview

DiploVault is a secure cloud-based file storage and management system developed using Java, JSP, Servlets, and MySQL. The project focuses on providing secure file upload, storage, search, and download functionalities using encryption techniques and role-based access control.

The system is designed to protect sensitive data using double-layer encryption with the Triple DES (3DES) algorithm. It also includes secure key-based file access, user authentication, clustering mechanisms, and efficient file management features. 

---

## Features

* Secure file upload and download
* Double-layer encryption using 3DES
* User authentication and authorization
* Role-Based Access Control (RBAC)
* Admin and User modules
* File clustering and management
* Secure key-based file access
* FTP-based encrypted file storage
* Email notification for file access keys
* Search files using keywords
* Activity monitoring and audit tracking
